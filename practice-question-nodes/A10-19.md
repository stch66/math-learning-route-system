# A10-19 10的整除规则

## Concept

**1. 10的整除规则：个位是0。为什么不用真的除也能先判断？**
A. 因为10能整除任何整数
B. 因为10是两位数
C. 因为只由个位决定，个位是0就能被10整除
D. 因为10的倍数都是两位数
**答案：C**
| 选项 | 错因 |
|---|---|
| A | 错误，10不能整除任何整数 |
| B | 没有解释规则本质 |
| C | 正确 |
| D | 10的倍数不一定是两位数 |

**2. 判断一个数能否被10整除，要看哪里？比如120这一类数。**
A. 看十位
B. 看所有数位之和
C. 看是不是10的倍数
D. 看个位
**答案：D**
| 选项 | 错因 |
|---|---|
| A | 看错数位 |
| B | 看错数位 |
| C | 没有说明具体判断方法 |
| D | 正确 |

## Model

**3. 把140写在数位表里，圈出需要看的数字。140能否被10整除？**
A. 能，个位是0
B. 能，1+4+0=5
C. 能，十位是4
D. 不能确定
**答案：A**
| 选项 | 错因 |
|---|---|
| A | 正确 |
| B | 看错数位 |
| C | 混淆了数字和规则 |
| D | 不理解规则 |

**4. 用分组图表示能整除时没有余数。140个物品每10个一组，分了几组？**
A. 分了130组
B. 分了14组
C. 分了150组
D. 分了140组
**答案：B**
| 选项 | 错因 |
|---|---|
| A | 误以为140-10就是组数 |
| B | 正确，140÷10=14 |
| C | 误以为加10就是组数 |
| D | 误以为每组1个 |

## Mental

**5. 快速判断：140能否被10整除；245能否被10整除；380能否被10整除。哪个正确？**
A. 140能，245能，380不能
B. 140能，245能，380能
C. 140不能，245能，380能
D. 140能，245不能，380能
**答案：D**
| 选项 | 错因 |
|---|---|
| A | 245和380判断反 |
| B | 245个位5漏判 |
| C | 140个位0漏判 |
| D | 245漏判 |

**6. 说出你的判断依据，不列竖式。判断245能否被10整除，依据是？**
A. 245是5的倍数，所以能
B. 个位是5，大于0，所以能
C. 个位是5，不是0，所以不能
D. 十位是4，所以能
**答案：C**
| 选项 | 错因 |
|---|---|
| A | 5的倍数不一定是10的倍数 |
| B | 误解规则 |
| C | 正确 |
| D | 看错数位 |

## Written

**7. 选择140和245，写出判断过程。哪个过程正确？**
A. 140个位是0，能被10整除；245个位是5，不能被10整除
B. 140个位是4，不能被10整除；245个位是2，能被10整除
C. 140个位是0，能被10整除；245个位是5，能被10整除
D. 140个位是4，能被10整除；245个位是5，能被10整除
**答案：A**
| 选项 | 错因 |
|---|---|
| A | 正确 |
| B | 140个位是0不是4，245个位是5不是2 |
| C | 245个位是5不是0 |
| D | 140个位是0不是4，245个位是5不是0 |

**8. 改错题：同学说“720 的十位是2，所以能被10整除”。怎样改最准确？**
A. 应该看个位，个位是0，所以720能被10整除
B. 应该看百位，百位是7，所以720不能被10整除
C. 十位是2确实能决定能否被10整除
D. 720是偶数，所以一定能被10整除
**答案：A**
| 选项 | 错因 |
|---|---|
| A | 正确，用个位0判断 |
| B | 看错数位 |
| C | 十位不能决定10的整除 |
| D | 偶数不一定能被10整除 |

## Application

**9. 十张一捆贴纸：编号260能不能按10个一组正好分完？哪个最符合10的整除规则？**
A. 能，因为个位是0
B. 能，因为260是10的26倍
C. 不能，因为260是偶数
D. 能，因为260÷10=26
**答案：A**
| 选项 | 错因 |
|---|---|
| A | 正确，用规则判断 |
| B | 结果对，但没有体现个位规则 |
| C | 偶数与10整除无关 |
| D | 结果对，但不是按规则判断 |

**10. 自己写一个能被10整除和一个不能整除的数。参考110和75，哪个正确？**
A. 能被10整除：110；不能被10整除：75
B. 能被10整除：75；不能被10整除：120
C. 能被10整除：120；不能被10整除：110
D. 能被10整除：105；不能被10整除：90
**答案：A**
| 选项 | 错因 |
|---|---|
| A | 正确，110个位0，75个位5 |
| B | 105个位5不能，90个位0能，互换了 |
| C | 120和110都能整除 |
| D | 90个位0能，105个位5不能，互换了 |
