# A19-3.2 在数轴上表示相反数 选择题（按五层设计）

## 概念层（2题）

**第1题**：在数轴上表示一对相反数时，两个点应满足什么？
- A. 一定相距 1 个单位（错：距离取决于数的大小）
- B. 都和 0 重合（错：只有 0 在原点）
- C. 关于 0 对称（对：相反数到 0 距离相等、方向相反）
- D. 都在 0 的右边（错：负数应在左边）

答案：C

---

**第2题**：点 P 在 0 左边 4 个单位，它的相反数点在哪里？
- A. 0 左边 8 个单位（错：方向和距离都错）
- B. 0 左边 4 个单位（错：还是原来的点）
- C. 0 右边 1/4 个单位（错：混成倒数）
- D. 0 右边 4 个单位（对：相反数在相反方向同样距离）

答案：D

---

## 模型层（2题）

**第3题**：要在数轴上画 2.5 和它的相反数，哪种描述正确？
- A. 2.5 在右边，-2.5 在左边，离 0 都是 2.5 个单位（对：位置关于 0 对称）
- B. 两个点都在 2 和 3 之间（错：-2.5 在负方向）
- C. 两个点都在 -2 和 -3 之间（错：2.5 在正方向）
- D. 一个点在 0 上，一个点在 2.5 上（错：相反数不是把一个点变成 0）

答案：A

---

**第4题**：数轴上 A 和 B 关于 0 对称，A=7，则 B 是多少？
- A. 0（错：0 是对称中心）
- B. -7（对：关于 0 对称的点表示相反数）
- C. 7（错：同一个位置不是对称点）
- D. 1/7（错：这是倒数）

答案：B

---

## 快速反应层（2题）

**第5题**：哪一组点最可能是一对相反数点？
- A. 3 和 1/3（错：不是相反方向）
- B. -3 和 -1/3（错：都在负方向）
- C. -3 和 3（对：到 0 距离相等，方向相反）
- D. -3 和 0（错：距离不相等）

答案：C

---

**第6题**：点 A 在 0 右边 6 格，点 B 在 0 左边 6 格。A+B 的结果应是多少？
- A. 12（错：把方向忽略了）
- B. -12（错：把两个都看成负数）
- C. 6（错：只看了一个点）
- D. 0（对：两个点表示 6 和 -6）

答案：D

---

## 规范层（2题）

**第7题**：哪种图示最能避免相反数画错？
- A. 先标 0，再在左右同距离处标两个点（对：对称中心和距离都明确）
- B. 先画两个点，再随便补 0（错：容易失去对称依据）
- C. 把两个点都画在右边（错：方向不相反）
- D. 只写数字，不画距离（错：看不出数轴关系）

答案：A

---

**第8题**：学生把 -5 和 5 画得一个离 0 两格、一个离 0 五格。错因是什么？
- A. 用了同一条数轴（错：同一条数轴是正确模型）
- B. 没有保持到 0 的距离相等（对：相反数点应关于 0 对称）
- C. 把 0 标出来了（错：标 0 是正确步骤）
- D. 把两个数写成整数（错：它们本来就是整数）

答案：B

---

## 应用层（2题）

**第9题**：温度 +8℃ 和 -8℃ 在温度计上怎样对应？
- A. 都在 0℃ 下方（错：+8℃ 在上方）
- B. 相距 8 格（错：它们相距 16 格）
- C. 分别在 0℃ 上下同样 8 格（对：它们关于 0℃ 对称）
- D. 都在 0℃ 上方（错：-8℃ 在下方）

答案：C

---

**第10题**：小车从 0 点向东 12 米和向西 12 米，两个终点的关系是什么？
- A. 是同一个终点（错：方向不同）
- B. 不能放在数轴上（错：直线运动可用数轴表示）
- C. 互为倒数位置（错：位置关系不是倒数）
- D. 表示相反数位置（对：同距离、反方向）

答案：D
