# A19-2.3 在数轴上标负整数 选择题（按五层设计）

## 概念层（2题）

**第1题**：负整数在标准数轴上通常位于 0 的哪一边？
- A. 左边（对：负方向通常向左）
- B. 右边（错：右边通常表示正数）
- C. 只能和 0 重合（错：只有 0 在原点）
- D. 不能表示（错：负整数可以在数轴上表示）

答案：A

---

**第2题**：标 -5 时，最准确的动作是哪项？
- A. 从 0 向右走 5 个单位（错：方向反了）
- B. 先找 5 再擦掉正号（错：位置应在相反方向）
- C. 把 -5 放在 0 和 1 之间（错：-5 不在这个区间）
- D. 从 0 向左走 5 个单位（对：负号表示负方向）

答案：D

---

## 模型层（2题）

**第3题**：数轴每格为 1，0 左边第 4 格是什么？
- A. -3（错：少数了一格）
- B. -1/4（错：把格数误作倒数）
- C. -4（对：向左每格减少 1）
- D. 4（错：漏看方向）

答案：C

---

**第4题**：-2 和 -6 都在 0 左边，哪一个离 0 更远？
- A. 无法判断（错：距离可由绝对值判断）
- B. -6（对：-6 到 0 距离为 6）
- C. -2（错：-2 到 0 距离为 2）
- D. 一样远（错：距离不同）

答案：B

---

## 快速反应层（2题）

**第5题**：快速读位置：点 A 在 0 左边 7 格，A 是多少？
- A. -7（对：左边 7 格对应 -7）
- B. 7（错：方向反了）
- C. -6（错：少数一格）
- D. 0（错：不在原点）

答案：A

---

**第6题**：点 B 在 -3 的左边一格，B 表示什么？
- A. -2（错：这是向右一格）
- B. 4（错：符号错了）
- C. 0（错：没有回到原点）
- D. -4（对：向左一格数值减少 1）

答案：D

---

## 规范层（2题）

**第7题**：把 -1、-2、-3、-4 标在数轴上，哪种顺序正确？
- A. 全部写在 0 右边（错：负整数应在左边）
- B. -1 和 -4 写在同一点（错：数值不同位置不同）
- C. 从左到右是 -4、-3、-2、-1（对：越往右数越大）
- D. 从左到右是 -1、-2、-3、-4（错：方向反了）

答案：C

---

**第8题**：同学把 -8 标在 +8 的位置，主要错因是什么？
- A. -8 不能在数轴上表示（错：负整数可以表示）
- B. 只看数字 8，忽略负号方向（对：负号决定在 0 左边）
- C. 单位长度太整齐（错：整齐不是问题）
- D. 0 写成原点（错：这一步是正确的）

答案：B

---

## 应用层（2题）

**第9题**：地下 3 层用数轴表示，最合理的位置是？
- A. -3，在 0 左边 3 个单位（对：地下方向常用负数）
- B. +3，在 0 右边 3 个单位（错：方向意义反了）
- C. 0，因为在楼里（错：地下 3 层不是地面）
- D. 1/3，因为有 3 层（错：层数不是分数关系）

答案：A

---

**第10题**：气温 -9℃ 比 -4℃ 在温度计上怎样？
- A. 更高，因为 9 比 4 大（错：只比较绝对值错了）
- B. 一样高（错：温度不同）
- C. 不能用数轴表示（错：温度计可看作数轴）
- D. 更低，位置更靠负方向（对：-9 小于 -4）

答案：D
