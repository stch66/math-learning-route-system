# A10-10 估算积

## Concept

**1. 估算积时，通常把因数约成整十数。58×23 可以约成多少乘多少？**
A. 约成 58×20
B. 约成 60×20
C. 约成 60×23
D. 约成 50×20
**答案：B**

**2. 为什么估算积能先判断答案数量级？哪个说法正确？**
A. 因为约成整十数后能快速估计积大约是几百还是几千
B. 因为估算积比精算更准确
C. 因为估算积不用看因数大小
D. 因为估算积一定是准确的
**答案：A**
| 选项 | 错因 |
|---|---|
| A | 正确 |
| B | 误解准确度 |
| C | 误解原理 |
| D | 误解准确度 |

## Model

**3. 画面积模型，把 72 看成接近 70，把 38 看成接近 40。估算面积大约是多少？**
A. 70×38=2660
B. 72×38=2736
C. 70×40=2800
D. 70×30=2100
**答案：C**

**4. 用方格图表示大约面积。哪个结果最符合整十估算？**
A. 约 1200，60×20=1200 差 134
B. 约 1380，60×23=1380 差 46
C. 约 1334，58×23=1334 最接近
D. 约 1200，60×20=1200 差 134，但方向相反
**答案：A**

## Mental

**5. 快速估算：64×31≈（　）。**
A. 1984
B. 1600
C. 1800
D. 2100
**答案：C**
| 选项 | 错因 |
|---|---|
| A | 精确值，不是估算 |
| B | 31约成25偏小 |
| C | 正确，64×31≈60×30=1800 |
| D | 估得偏大 |

**6. 判断精确积是几百还是几千？哪个正确？**
A. 几千，因为 58×23 比 1000 大
B. 几千，因为 58 和 23 都是两位数
C. 几千，因为 60×20=1200，是几千的数量级
D. 几千，因为 50×20=1000
**答案：C**
| 选项 | 错因 |
|---|---|
| A | 误解数量级 |
| B | 误解数量级 |
| C | 正确 |
| D | 误解数量级 |

## Written

**7. 写估算步骤：86≈（　），42≈（　），积约（　）。**
A. 86≈80，42≈40，积约 3200
B. 86≈80，42≈40，积约 3200（未写单位）
C. 86≈90，42≈40，积约 3600
D. 86≈90，42≈45，积约 4050
**答案：C**
| 选项 | 错因 |
|---|---|
| A | 86约得偏小 |
| B | 86约得偏小 |
| C | 正确 |
| D | 42不宜约成45 |

**8. 再算精确值，检查估算是否合理。哪个结果在合理范围内？**
A. 58×23=1334，离 1200 很远，不合理
B. 58×23=1334，在 1200 附近，合理
C. 58×23=1334，和 1200 差 134，很合理
D. 58×23=1334，比 1200 大很多，不合理
**答案：B**
| 选项 | 错因 |
|---|---|
| A | 判断反 |
| B | 正确 |
| C | 判断错 |
| D | 判断反 |

## Application

**9. 每箱数量乘箱数：每箱约 47 个，共 19 箱，大约多少个？**
A. 约 900 个，只约了一个因数
B. 约 893 个，精算
C. 约 1000 个，47×19≈50×20=1000
D. 约 800 个，进位错误
**答案：C**

**10. 自己编一道估算积问题。哪个估算最合理？**
A. 62×34≈60×30=1800
B. 62×34≈60×35=2100
C. 62×34≈60×34=2040
D. 62×34≈62×34=2108
**答案：A**
| 选项 | 错因 |
|---|---|
| A | 正确 |
| B | 只改一个 |
| C | 精确值 |
| D | 精确值 |
