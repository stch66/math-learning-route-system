# A10-1 加法交换律

## Concept

**1. 加法交换律说的是交换两个加数的位置，和不变。27+53 和 53+27 为什么一样？**
A. 交换的是加数的位置，结果不受位置影响
B. 因为先算了 27+53 才能算 53+27
C. 因为两个数都是整十数
D. 因为 27 比 53 小，所以 53+27 更大
**答案：A**
| 选项 | 错因 |
|---|---|
| A | 正确 |
| B | 误以为加数大小决定和的大小，忽略了加法本质 |
| C | 混淆了运算顺序的因果关系 |
| D | 误解了交换律，只看数字特征 |

---

**2. 请说清：变的是（　），不变的是（　）。**
A. 变的是和，不变的是顺序
B. 变的是数字本身，不变的是结果
C. 变的是加号，不变的是数字
D. 变的是顺序，不变的是和
**答案：D**
| 选项 | 错因 |
|---|---|
| A | 说反了，交换律是「和不变，顺序变」 |
| B | 颠倒了变化和不变的量 |
| C | 混淆了变化的量 |
| D | 正确，变的是加数顺序，不变的是和 |

---

## Model

**3. 画两盒彩笔：一盒 36 支，一盒 59 支；再把左右位置交换。总数会变吗？**
A. 总数变大，因为换了位置
B. 总数变小，因为换了位置
C. 总数不确定，要看先数哪盒
D. 总数不变，还是 95 支
**答案：D**
| 选项 | 错因 |
|---|---|
| A | 误以为位置影响数量，不理解加法的次序不变性 |
| B | 误以为位置影响数量，不理解加法的次序不变性 |
| C | 误以为计数顺序会影响结果，没有理解加法的本质 |
| D | 正确，加法交换律保证交换后和不变 |

---

**4. 用数轴或条形图表示两个算式的总长度一样。哪个最能说明 48+31 和 31+48 相等？**
A. 数轴上两个算式起点到终点的总长度相同
B. 条形图的高度不同
C. 两个算式方向不同，所以长度不同
D. 数轴上 48 比 31 大，所以 48+31 更长
**答案：A**
| 选项 | 错因 |
|---|---|
| A | 正确 |
| B | 只比较了加数大小，没有理解加法叠加原理 |
| C | 混淆了条形图的表示方式 |
| D | 不理解数轴叠加原理，混淆方向和长度 |

## Mental

**5. 不用重算，已知 67+28=95，马上说 28+67=（　）。**
A. 85
B. 73
C. 95
D. 39
**答案：C**
| 选项 | 错因 |
|---|---|
| A | 误以为交换后和会变大（85>95） |
| B | 进位错误 |
| C | 正确，直接应用交换律 |
| D | 把28+67误算成67-28 |

---

**6. 判断 54+27 和 27+54 是否相等，并说明。**
A. 相等，但不知道为什么
B. 不相等，因为数字位置变了
C. 相等，因为交换律
D. 不一定，要看哪个数在前
**答案：C**
| 选项 | 错因 |
|---|---|
| A | 不理解交换律本质，死记硬背 |
| B | 不理解交换律，认为位置决定结果 |
| C | 正确，理解交换律的本质 |
| D | 不理解交换律，认为先后顺序有影响 |

## Written

**7. 写一组交换律算式，并用等号连接。哪组最合适？**
A. 45+82=45+82（没有真正交换）
B. 45+82=82+45=127
C. 45+82=127
D. 45+82=82+45
**答案：B**
| 选项 | 错因 |
|---|---|
| A | 正确展示交换律等式 |
| B | 只写了结果，没有体现交换律 |
| C | 没有体现交换，写的是同一个算式 |
| D | 用等号连接三个表达式，结构错误 |

---

**8. 把 45+82 改写成 82+45，写一句"为什么可以"。哪句最准确？**
A. 交换律：两个数相加，位置交换和不变
B. 因为 45+82 比较难算
C. 因为 82+45 看起来更整齐
D. 因为老师要求这样写
**答案：A**
| 选项 | 错因 |
|---|---|
| A | 正确 |
| B | 不是数学理由，只是计算偏好 |
| C | 不是数学理由，只是视觉偏好 |
| D | 不是数学理由，只是服从权威 |

## Application

**9. 两盒彩笔换左右位置：先数左盒还是右盒，总数会变吗？请列式说明。**
A. 先数左盒总数变大
B. 先数右盒总数变小
C. 总数不变，36+59=95 或 59+36=95
D. 要看哪盒摆在上面
**答案：C**
| 选项 | 错因 |
|---|---|
| A | 误以为先数会影响结果 |
| B | 误以为先数会影响结果 |
| C | 正确，加法交换律保证交换后和不变 |
| D | 误以为物理位置会影响数量 |

---

**10. 自己编一道可以用加法交换律检查的题。哪道符合？**
A. 小明有 24 支笔，掉了 7 支，还剩多少？
B. 小明有 24 支笔，是小红的 4 倍，小红有几支？
C. 小明有 24 支笔，小红有 31 支笔，两人一共有多少支？
D. 小明有 24 支笔，又买了 7 支，现在有几支？
**答案：C**
| 选项 | 错因 |
|---|---|
| A | 正确，两数相加可用交换律交换加数顺序检查 |
| B | 减法题，交换律不适用 |
| C | 乘法（包含）关系题，交换律不适用 |
| D | 先加后减，顺序有影响，交换律不直接适用 |
