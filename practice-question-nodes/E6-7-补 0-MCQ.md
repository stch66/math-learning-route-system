# E6-7 补 0 选择题（按五层设计）

## 概念层（2题）

**第1题**：为什么 3.5 可以写成 3.50？
- A. 3.50 比 3.5 大 10 倍（错：不符合题意）
- B. 3.50 比 3.5 小（错：不符合题意）
- C. 补 0 后变成 350（错：不符合题意）
- D. 末尾补 0 不改变小数大小（对：符合题意）

答案：D

---

**第2题**：补 0 的主要用途是什么？
- A. 帮助小数加减时位值对齐（对：符合题意）
- B. 让每个小数都变大（错：不符合题意）
- C. 让分数变成整数（错：不符合题意）
- D. 把小数点删掉（错：不符合题意）

答案：A

---

## 笔算层（2题）

**第7题**：规范补 0，哪一行正确？
