# A10-24 用最大公因数做化简准备

## Concept

**1. 化简分数前，先找分子15和分母20的最大公因数。为什么？**
A. 因为最大公因数可以让分数变大
B. 因为找到最大公因数后，同时除以它，分数大小不变但更简洁
C. 因为最大公因数可以让分母变小
D. 因为老师要求这样做
**答案：B**
| 选项 | 错因 |
|---|---|
| A | 误解作用 |
| B | 正确 |
| C | 误解作用 |
| D | 非数学理由 |

**2. 如果同时除以最大公因数，分数大小会变吗？**
A. 会变小
B. 不会，分子分母同时除以同一个数，分数大小不变
C. 会变大
D. 会变成整数
**答案：B**
| 选项 | 错因 |
|---|---|
| A | 误解 |
| B | 正确 |
| C | 误解 |
| D | 误解 |

## Model

**3. 画18/24的条形图，想一想可以按几等份重新合并。18/24化简后等于几分之几？**
A. 等于3/4，因为同时除以6
B. 等于6/8，因为同时除以3
C. 等于3/4，因为同时除以4
D. 等于9/12，因为同时除以2
**答案：A**
| 选项 | 错因 |
|---|---|
| A | 正确，18÷6=3，24÷6=4 |
| B | 还能继续化简，不是最简 |
| C | 18÷4不是整数 |
| D | 还能继续化简，不是最简 |

**4. 列出18和24的因数集合。哪个是正确的？**
A. 18的因数：1、2、3、6、9、18；24的因数：1、2、3、4、6、8、12、24；最大公因数：6
B. 18的因数：1、2、3、6、9、18；24的因数：1、2、3、4、6、8、12、24；最大公因数：4
C. 18的因数：1、2、3、6、18；24的因数：1、2、3、4、6、8、12、24；最大公因数：6
D. 18的因数：1、2、3、6、9、18；24的因数：1、2、3、4、6、8、24；最大公因数：6
**答案：A**
| 选项 | 错因 |
|---|---|
| A | 正确 |
| B | 4不是18的因数 |
| C | 18漏了9 |
| D | 24漏了12 |
