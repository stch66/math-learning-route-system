# A19-4.3 正数、负数和0比较 选择题（按五层设计）

## 概念层（2题）

**第1题**：比较正数、负数和 0 时，哪条分界最重要？
- A. 0 是正负数的分界点（对：正数大于 0，负数小于 0）
- B. 所有数都比 0 小（错：正数大于 0）
- C. 负号可以忽略（错：负号决定方向）
- D. 0 是最大的数（错：正数大于 0）

答案：A

---

**第2题**：在 -3、0、2 中，最大的是谁？
- A. 三个一样大（错：数值不同）
- B. 2（对：正数大于 0 和负数）
- C. 0（错：0 小于 2）
- D. -3（错：负数最小）

答案：B

---

## 模型层（2题）

**第3题**：数轴上 -5、0、4 的位置从左到右是？
- A. 0、-5、4（错：0 和 -5 顺序错）
- B. -5、4、0（错：4 和 0 顺序错）
- C. -5、0、4（对：负数在左，0 居中，正数在右）
- D. 4、0、-5（错：方向反了）

答案：C

---

**第4题**：若红队得 +6 分，蓝队得 0 分，绿队得 -2 分，排名第一是谁？
- A. 蓝队（错：0 小于 +6）
- B. 绿队（错：-2 小于 0）
- C. 三队并列（错：分数不同）
- D. 红队（对：+6 最大）

答案：D

---

## 快速反应层（2题）

**第5题**：快速判断：哪个数最小？ -1、0、1、-4。
- A. -4（对：负数中离 0 更远的更小）
- B. -1（错：-4 更小）
- C. 0（错：负数小于 0）
- D. 1（错：正数最大）

答案：A

---

**第6题**：同分母分数中，哪个比较符号正确？
- A. -1=1（错：相反数不相等）
- B. -2<0（对：负数小于 0）
- C. 0< -2（错：方向反了）
- D. 3<0（错：正数大于 0）

答案：B

---

## 规范层（2题）

**第7题**：哪一行说明 -7、0、5 的大小关系最完整？
- A. 0<-7<5（错：0 和 -7 顺序错）
- B. -7<5<0（错：5 和 0 顺序错）
- C. -7<0<5（对：按负数、0、正数排序）
- D. 5<0<-7（错：完全反向）

答案：C

---

**第8题**：学生把 -6 判成比 0 大，最可能看错了什么？
- A. 6 是自然数（错：题目中的数是 -6）
- B. 0 是原点（错：这点本身正确）
- C. 比较符号必须写成 =（错：不相等）
- D. 负号表示在 0 左边（对：负数应小于 0）

答案：D

---

## 应用层（2题）

**第9题**：海拔 -20 米、0 米、15 米，最高位置是哪一个？
- A. 15 米（对：正海拔最高）
- B. 0 米（错：低于 15 米）
- C. -20 米（错：在海平面下）
- D. 三者一样高（错：位置不同）

答案：A

---

**第10题**：账户变化 -8 元、+3 元、0 元，按“亏到赚”的顺序应怎样排？
- A. -8 元、+3 元、0 元（错：+3 和 0 顺序错）
- B. -8 元、0 元、+3 元（对：从小到大排列）
- C. +3 元、0 元、-8 元（错：从大到小）
- D. 0 元、-8 元、+3 元（错：0 和负数顺序错）

答案：B
