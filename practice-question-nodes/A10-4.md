# A10-4 乘法结合律

## Concept

**1. 乘法结合律改变的是先把哪两个因数相乘。6×7×5 先算哪两个更方便？**
A. 先算 6×7=42，再×5=210
B. 随便先算哪两个，结果一样
C. 先算 6×5=30，再×7=210
D. 先算 7×5=35，再×6=210
**答案：C**
| 选项 | 错因 |
|---|---|
| A | 虽然结果对，但42×5不如先凑30直观 |
| B | 虽然结果一样，但未体现"更方便" |
| C | 正确，先算6×5=30，再乘7更容易 |
| D | 虽然结果一样，但未体现"更方便" |

**2. 为什么先算 8×5 再乘 9 会更快？**
A. 因为 8×5=40，40×9 更容易算
B. 因为老师要求先算前两个
C. 因为 8×9×5 不能改变顺序
D. 因为 9×5 比 8×5 大
**答案：A**
| 选项 | 错因 |
|---|---|
| A | 正确，40×9是整十数后续计算更简便 |
| B | 非数学理由 |
| C | 顺序可改 |
| D | 只比较大小，没有说明计算便利性 |

## Model

**3. 画包装图：每小包 3 个，7 小包成一组，再有 5 组。总共有多少个？**
A. 3×7×5=15×5=75
B. 3×7×5=105，先算哪两个都一样
C. 3×7×5=105，共 105 个
D. 3×7×5=3×7×5=105
**答案：C**
| 选项 | 错因 |
|---|---|
| A | 算术错 |
| B | 结果相同但无体现 |
| C | 正确 |
| D | 无括号等价 |

**4. 用括号表示两种分组：(3×7)×5 和 3×(7×5)。哪个说法正确？**
A. (3×7)×5 比 3×(7×5) 大
B. 两种分组结果不同
C. 3×(7×5) 比 (3×7)×5 大
D. 两种分组结果都是 105，积不变
**答案：D**
| 选项 | 错因 |
|---|---|
| A | 结果不同错 |
| B | 结果不同错 |
| C | 不符 |
| D | 正确 |

## Mental

**5. 快速算 4×9×25=（　），说出先算哪一对。**
A. 900
B. 700
C. 100
D. 800
**答案：A**
| 选项 | 错因 |
|---|---|
| A | 正确，先算4×25=100，再乘9 |
| B | 算错 |
| C | 只算了4×25 |
| D | 算错 |

**6. 判断 4×6×5 先算 4×6 是否更方便。哪个说法正确？**
A. 不方便，因为 4×6 比 6×5 小
B. 更方便，因为 4×6=24，24×5=120
C. 不方便，因为 4×6×5 不能改变顺序
D. 随便先算哪两个都一样
**答案：B**
| 选项 | 错因 |
|---|---|
| A | 误解"方便"含义 |
| B | 正确，24×5是整十数 |
| C | 乘法结合律允许改变运算顺序 |
| D | 不同顺序计算难度不同 |

## Written

**7. 写出用乘法结合律简算的两步。哪步最准确？**
A. 4×6×5=4×(6×5)=4×30=120
B. 4×6×5=4×6×5=120
C. 4×6×5=(4×6)×5=24×5=120
D. 4×6×5=(4×5)×6=20×6=120
**答案：C**
| 选项 | 错因 |
|---|---|
| A | 计算顺序错误 |
| B | 没有体现乘法结合律 |
| C | 正确 |
| D | 虽然结果对但不是最简便的结合方式 |

**8. 圈出括号位置，说明积不变。哪个括号位置是正确的？**
A. (4×6)×5 和 4×(6×5) 结果不同
B. (4×6)×5 或 4×(6×5)
C. 括号必须加在 4×5 周围
D. (4×6)×5 和 4×6×5 一样
**答案：B**
| 选项 | 错因 |
|---|---|
| A | 误以为不同括号位置结果不同 |
| B | 正确 |
| C | 没有数学依据 |
| D | 没有体现括号的作用 |

## Application

**9. 包装盒彩带组合：按不同打包顺序数，总数会变吗？**
A. 总数会变
B. 按 (3×7)×5 更多
C. 总数不变，(3×7)×5=3×(7×5)=105
D. 按 3×(7×5) 更多
**答案：C**
| 选项 | 错因 |
|---|---|
| A | 误解结合律 |
| B | 误以为不同顺序结果不同 |
| C | 正确 |
| D | 误以为不同顺序结果不同 |

**10. 自己编一道能先凑 10 或 100 的连乘题。哪道符合？**
A. 6×6×6（只有216）
B. 3×4×5（不能凑整）
C. 5×7×2（没有整十整百）
D. 4×25×3（先算4×25=100，再乘3=300）
**答案：D**
| 选项 | 错因 |
|---|---|
| A | 正确，但没有体现凑整便利性 |
| B | 不能凑整 |
| C | 无整百 |
| D | 正确，4×25=100，后续计算更方便 |
