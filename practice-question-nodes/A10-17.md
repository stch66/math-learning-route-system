# A10-17 2的整除规则

## Concept

**1. 判断一个数能不能被2整除，最直接看哪里？**
A. 看十位
B. 看所有数位的和
C. 看个位是不是0、2、4、6、8
D. 看这个数是不是比2大
**答案：C**
| 选项 | 错因 |
|---|---|
| A | 看错数位 |
| B | 混淆了3的整除规则 |
| C | 正确 |
| D | 只看大小，不能判断整除 |

---

**2. 下面哪个数一定能被2整除？比如先观察58这一类数的个位。**
A. 37
B. 58
C. 91
D. 105
**答案：B**
| 选项 | 错因 |
|---|---|
| A | 个位是7，不能被2整除 |
| B | 正确，个位是8 |
| C | 个位是1，不能被2整除 |
| D | 个位是5，不能被2整除 |

---

## Model

**3. 把84写在数位表里，圈出需要看的数字。84能否被2整除？**
A. 不能确定
B. 能，8+4=12
C. 能，十位是8
D. 能，个位是4
**答案：D**
| 选项 | 错因 |
|---|---|
| A | 不能确定错，个位是4可以判断 |
| B | 看错数位 |
| C | 混淆了3的整除规则 |
| D | 正确 |

**4. 用分组图表示能整除时没有余数。84个物品每2个一组，分了几组？**
A. 分了42组
B. 分了82组
C. 分了86组
D. 分了84组
**答案：A**
| 选项 | 错因 |
|---|---|
| A | 正确，84÷2=42 |
| B | 误以为每组分2个 |
| C | 误以为加2就是分组数 |
| D | 误以为每组1个 |

## Mental

**5. 快速判断：84能否被2整除；57能否被2整除；238能否被2整除。哪个正确？**
A. 84不能，57能，238能
B. 84能，57不能，238能
C. 84能，57能，238不能
D. 84能，57能，238能
**答案：B**
| 选项 | 错因 |
|---|---|
| A | 84和238判断反，57也判断反 |
| B | 正确 |
| C | 57个位是7，不能被2整除 |
| D | 57漏判 |

**6. 说出你的判断依据，不列竖式。判断59能否被2整除，依据是？**
A. 十位是5，所以不能
B. 个位是9，不是偶数，所以不能
C. 59是奇数，所以不能
D. 个位是9，但9是奇数，所以不能
**答案：B**
| 选项 | 错因 |
|---|---|
| A | 看错数位 |
| B | 正确，判断2的整除看个位 |
| C | 结果对，但没有直接说明个位规则 |
| D | 结果对，但表述绕了一步 |

## Written

**7. 选择84和57，写出判断过程。哪个过程正确？**
A. 84个位是4，能被2整除；57个位是7，不能被2整除
B. 84十位是8，不能被2整除；57个位是7，不能被2整除
C. 84个位是4，能被2整除；57个位是7，也能被2整除
D. 84个位是4，能被2整除；57个位是7，能被2整除
**答案：A**
| 选项 | 错因 |
|---|---|
| A | 正确 |
| B | 判断84时看错数位 |
| C | 57个位是7不是偶数但判为能 |
| D | 57个位是7不是偶数 |

**8. 改错题：同学说“312 的百位是 3，所以能被 2 整除”。怎样改最准确？**
A. 应该看个位，个位是2，所以312能被2整除
B. 应该看十位，十位是1，所以312不能被2整除
C. 百位是3也可以判断，所以原说法完全正确
D. 312比2大，所以一定能被2整除
**答案：A**
| 选项 | 错因 |
|---|---|
| A | 正确，改成看个位规则 |
| B | 看错数位 |
| C | 仍然看百位，理由错 |
| D | 只看大小不能判断 |

## Application

**9. 偶数门票编号：编号146能不能按2个一组正好分完？**
A. 能，因为146÷2=73
B. 能，因为146是偶数（个位是6）
C. 不能，因为146是偶数
D. 能，因为146=140+6
**答案：B**
| 选项 | 错因 |
|---|---|
| A | 结果对，但这里没有体现按个位判断的规则 |
| B | 正确，用2的整除规则判断 |
| C | 混淆了奇偶性和整除的关系 |
| D | 没有直接说明整除关系 |

**10. 自己写一个能被2整除和一个不能整除的数。参考88和95，哪个正确？**
A. 能被2整除：88；不能被2整除：95
B. 能被2整除：88；不能被2整除：98
C. 能被2整除：87；不能被2整除：98
D. 能被2整除：85；不能被2整除：59
**答案：A**
| 选项 | 错因 |
|---|---|
| A | 正确，88个位8是偶数，95个位5不是 |
| B | 98个位8是偶数，也能被2整除 |
| C | 87个位7不是偶数，不能被2整除 |
| D | 85个位5不是偶数，不能被2整除；59个位9不是偶数，不能被2整除 |
