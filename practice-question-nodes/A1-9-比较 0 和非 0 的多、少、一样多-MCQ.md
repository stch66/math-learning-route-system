# A1-9 比较 0 和非 0 的多、少、一样多 选择题（按五层设计）

## 概念层（2题）

**第1题**：比较多少时，先要做什么？

- A. 0
- B. >
- C. 先数清两组数量
- D. <

答案：C

---

**第2题**：比较 0 和 4，哪个更多？

- A. 0
- B. >
- C. <
- D. 4

答案：D

---

## 模型层（2题）

**第3题**：比较 0 和 1，哪个更少？

- A. 0
- B. >
- C. <
- D. 0?1

答案：A

---

**第4题**：用符号表示 0 和 5，应填什么？

- A. 0
- B. <
- C. >
- D. <?1

答案：B

---

## 口算层（2题）

**第5题**：口答：0 < 3 是否正确？

- A. 0
- B. >
- C. 正确
- D. <

答案：C

---

**第6题**：如果两个数量相同，应说什么？

- A. 0
- B. >
- C. <
- D. 一样多

答案：D

---

## 笔算层（2题）

**第7题**：哪一项正确比较 2 和 0？

- A. 2 > 0
- B. 2 < 0
- C. 2 = 0
- D. 0 > 2

答案：A

---

**第8题**：检查比较时，要看什么？

- A. 0
- B. 两边数量
- C. >
- D. <

答案：B

---

## 应用层（2题）

**第9题**：两盘分别有 6 个和 0 个，哪盘更多？

- A. 6 个那盘
- B. 0 个那盘
- C. 一样多
- D. 无法比较

答案：A

---

**第10题**：哪句话适合说明多、少、一样多？

- A. 0
- B. >
- C. <
- D. 数清后再比较

答案：D

---
