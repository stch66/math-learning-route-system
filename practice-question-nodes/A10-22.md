# A10-22 列因数求最大公因数

## Concept

**1. 最大公因数是两个数共同因数里最大的一个。18和24的共同因数有哪些？**
A. 1、2、3、6、18
B. 1、2、3、4、6
C. 1、2、3、6
D. 1、2、3、4、6、12
**答案：C**
| 选项 | 错因 |
|---|---|
| A | 多了18 |
| B | 多了4，4不是18的因数 |
| C | 正确 |
| D | 多了12，12不是18的因数 |

**2. 为什么要先分别列因数，再找共同的？哪个说法正确？**
A. 因为两个数的因数太多
B. 因为这样比较快
C. 因为老师要求这样做
D. 因为不分别列出来就找不到共同因数，也无法比较哪个最大
**答案：D**
| 选项 | 错因 |
|---|---|
| A | 非数学理由 |
| B | 非数学理由 |
| C | 非数学理由 |
| D | 正确 |

## Model

**3. 画两个因数集合圈，分别写20和30的因数。哪个因数列表正确？**
A. 20的因数：1、2、4、5、10、20；30的因数：1、2、3、5、6、10、15、30
B. 20的因数：1、2、4、5、20；30的因数：1、2、3、5、6、10、30
C. 20的因数：1、2、4、5、10、20；30的因数：1、2、5、10、15、30
D. 20的因数：1、2、5、10、20；30的因数：1、2、3、5、6、10、15、30
**答案：A**
| 选项 | 错因 |
|---|---|
| A | 正确 |
| B | 两边都漏了因数 |
| C | 30漏了3和6 |
| D | 20漏了4 |

**4. 把共同因数放在重叠区域。16和28的共同因数有几个？**
A. 4个：1、2、4、16
B. 3个：1、2、4
C. 2个：1、2
D. 1个：1
**答案：B**
| 选项 | 错因 |
|---|---|
| A | 多了16 |
| B | 正确 |
| C | 少了4 |
| D | 少了2和4 |

| A | 正确 |
| B | 两边都漏了因数 |
| C | 42漏了14和21 |

**6. 35和49的最大共同因数是多少？**

## Written

**7. 完整列因数求36和48的最大公因数。哪个步骤正确？**
A. 36的因数：1、2、3、4、6、9、12、18、36；48的因数：1、2、3、4、6、8、12、16、24、48；共同因数：1、2、3、4、6、12；最大公因数：6
B. 36的因数：1、2、3、4、6、9、12、18、36；48的因数：1、2、3、4、6、8、12、16、24、48；共同因数：1、2、3、4、6、12；最大公因数：12
C. 36的因数：1、2、3、4、6、9、18、36；48的因数：1、2、3、4、6、8、16、24、48；共同因数：1、2、3、4、6；最大公因数：6
D. 36的因数：1、2、3、4、6、9、12、18、36；48的因数：1、2、3、4、6、8、12、16、24、48；共同因数：1、2、3、4、6、12；最大公因数：24
**答案：B**
| 选项 | 错因 |
|---|---|
| A | 不是最大 |
| B | 正确 |
| C | 漏掉12 |
| D | 24不是36的因数 |

**8. 用最大公因数检查能否把两盒彩带剪成同样长且不浪费。30cm和45cm彩带剪成同样长，最大每段几厘米？**
A. 5cm，因为5是共同因数
B. 15cm，因为30和45的最大公因数是15
C. 10cm，因为10是30的因数
D. 45cm，因为45最大
**答案：B**
| 选项 | 错因 |
|---|---|
| A | 不是最大 |
| B | 正确 |
| C | 10不是45的因数 |
| D | 45不是30的因数 |

## Application

**9. 两盒彩带剪成同样长：45厘米和60厘米彩带剪成同样长且不浪费，最长每段几厘米？**
A. 5厘米
B. 10厘米
C. 15厘米
D. 30厘米
**答案：C**
| 选项 | 错因 |
|---|---|
| A | 不是最大 |
| B | 不是最大 |
| C | 正确，15×3=45，15×4=60 |
| D | 不是45的因数 |

**10. 说说最大公因数怎样帮助化简分数。哪个说法正确？**
A. 最大公因数帮助找最小公倍数
B. 最大公因数没有实际用途
C. 最大公因数帮助算加减法
D. 找到分子分母的最大公因数，同时除以它，分数大小不变但更简洁
**答案：D**
| 选项 | 错因 |
|---|---|
| A | 无关 |
| B | 无关 |
| C | 无关 |
| D | 正确 |
