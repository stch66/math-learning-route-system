# A10-12 找因数

## Concept

**1. 因数是能整除 45 的数。什么数乘什么数可以得到 45？**
A. 1×45、3×15
B. 只要两个数相乘等于 45，这两数都是 45 的因数
C. 45÷1=45、45÷3=15
D. 1×45、3×15、5×9
**答案：D**
| 选项 | 错因 |
|---|---|
| A | 不完整，漏掉了5×9 |
| B | 说法对但没有列出具体因数对 |
| C | 用除法检查可以，但不完整 |
| D | 正确，完整列出所有因数对 |

---

**2. 找因数时为什么要成对找？哪个说法正确？**
A. 因为老师要求这样找
B. 因为因数必须成对出现
C. 因为成对找比较快
D. 因为成对找不会遗漏，且找到 √n 之后就不用再找
**答案：D**
| 选项 | 错因 |
|---|---|
| A | 非数学理由 |
| B | 表述不严密，容易漏掉平方数这种特殊配对 |
| C | 只说快，没有说明为什么不遗漏 |
| D | 正确 |

---

## Model

**3. 用 36 个方块摆长方形，记录每种边长。36 可以摆成几种不同长方形？**
A. 4 种：1×36、2×18、3×12、4×9
B. 5 种：1×36、2×18、3×12、4×9、6×6
C. 6 种：1×36、2×18、3×12、4×9、6×6、9×4
D. 3 种：1×36、2×18、3×12
**答案：B**
| 选项 | 错因 |
|---|---|
| A | 漏掉了6×6 |
| B | 正确，长宽顺序不同算同一种 |
| C | 9×4和4×9重复 |
| D | 漏掉了4×9和6×6 |

**4. 画因数配对表：1×40，2×（　）。下一个正确填写是？**
A. 9
B. 15
C. 20
D. 40
**答案：C**
| 选项 | 错因 |
|---|---|
| A | 2×9=18，不是40 |
| B | 2×15=30，不是40 |
| C | 正确，2×20=40 |
| D | 2×40=80，不是40 |

## Mental

**5. 快速判断：3、4、6、9 哪些是 54 的因数？**
A. 3 和 4 是，6 和 9 不是
B. 3 和 6 是，4 和 9 不是
C. 3、6、9 是，4 不是
D. 3、4、6、9 都是
**答案：C**
| 选项 | 错因 |
|---|---|
| A | 把4误判成因数，漏掉6和9 |
| B | 漏掉9 |
| C | 正确，54÷3、÷6、÷9都能整除，÷4有余数 |
| D | 4不是54的因数 |

**6. 说出 72 的两个因数对。哪组正确？**
A. 2×36 和 3×25
B. 1×72 和 4×18
C. 6×11 和 8×9
D. 1×72 和 4×16
**答案：B**
| 选项 | 错因 |
|---|---|
| A | 3×25=75，不是72 |
| B | 正确 |
| C | 6×11=66，不是72 |
| D | 4×16=64，不是72 |

## Written

**7. 按从小到大列出 60 的全部因数。哪个排列正确？**
A. 1、2、3、4、5、6、10、12、15、20、60
B. 1、2、3、4、5、6、10、12、15、20、30、60
C. 1、2、3、4、5、6、8、10、12、15、20、30、60
D. 1、2、3、5、6、10、12、15、30、60
**答案：B**
| 选项 | 错因 |
|---|---|
| A | 漏掉30 |
| B | 正确 |
| C | 8不是60的因数 |
| D | 漏掉4和20 |

**8. 用除法检查一个你不确定的因数。检查 7 是不是 52 的因数，哪个正确？**
A. 52÷7=7 余 3，所以 7 是因数
B. 52÷7=7，7 是 52 的因数
C. 52÷7=7 余 3，7 不是 52 的因数
D. 52÷7=7 余 3，7 可能是因数
**答案：C**
| 选项 | 错因 |
|---|---|
| A | 有余数就不是因数 |
| B | 忽略了余数3 |
| C | 正确 |
| D | 有余数就不是因数 |

## Application

**9. 48 个方块摆长方形：可以摆成哪些不同长方形？**
A. 1×48、2×24、3×16、4×12、6×8、8×6
B. 1×48、2×24、3×16
C. 1×48、2×24、3×16、4×12、6×8
D. 1×48、2×24、3×16、5×9
**答案：C**
| 选项 | 错因 |
|---|---|
| A | 8×6和6×8重复 |
| B | 漏掉4×12和6×8 |
| C | 正确，5种不同长方形 |
| D | 5×9=45，不是48 |

**10. 说说找因数怎样帮助以后化简分数。哪个说法正确？**
A. 找因数能帮助做乘法
B. 找因数可以帮助算加减法
C. 找因数可以知道分数大
D. 找到分子分母的公因数，可以同时除以它来化简
**答案：D**
| 选项 | 错因 |
|---|---|
| A | 无关 |
| B | 无关 |
| C | 无关 |
| D | 正确 |
