# A10-23 列倍数求最小公倍数

## Concept

**1. 最小公倍数是两个数共同倍数里最小的一个。6和8什么时候第一次同时出现？**
A. 24，因为24是6的倍数
B. 48，因为48是6和8的共同倍数
C. 24，因为6和8的最小公倍数是24
D. 16，因为16是8的倍数
**答案：C**
| 选项 | 错因 |
|---|---|
| A | 不是最小 |
| B | 不是最小 |
| C | 正确 |
| D | 不是共同倍数 |

**2. 为什么要列倍数而不是列因数？哪个说法正确？**
A. 因为最小公倍数是最大的共同倍数
B. 因为最小公倍数是共同倍数里最小的
C. 因为因数不够用
D. 因为倍数更好找
**答案：B**
| 选项 | 错因 |
|---|---|
| A | 误解定义 |
| B | 正确 |
| C | 非数学理由 |
| D | 非数学理由 |

## Model

**3. 画两条倍数数轴：7的倍数和9的倍数。哪个是7的前几个倍数？**
A. 7、14、21、28、35……
B. 7、14、21、28、49……
C. 7、14、21、28、42……
D. 7、14、21、28、36……
**答案：A**
| 选项 | 错因 |
|---|---|
| A | 正确 |
| B | 49是第7个，不是第5个 |
| C | 42是第6个，不是第5个 |
| D | 36不是7的倍数 |

**4. 圈出第一个共同出现的数。7和9的最小公倍数是多少？**
A. 63
B. 21
C. 72
D. 49
**答案：A**
| 选项 | 错因 |
|---|---|
| A | 正确 |
| B | 不是9的倍数 |
| C | 是9的倍数但不是7的倍数 |
| D | 不是9的倍数 |

## Mental

**5. 列出4的倍数和10的倍数，找到最小公倍数。哪个正确？**
A. 4的倍数：4、8、12、16、20……；10的倍数：10、20、30……；最小公倍数：20
B. 4的倍数：4、8、12、16、20……；10的倍数：10、20、30……；最小公倍数：10
C. 4的倍数：4、8、12、16、20……；10的倍数：10、20、30……；最小公倍数：40
D. 4的倍数：4、8、12、16、20……；10的倍数：10、20、30……；最小公倍数：8
**答案：A**
| 选项 | 错因 |
|---|---|
| A | 正确 |
| B | 不是最小 |
| C | 不是最小 |
| D | 不是最小 |

**6. 判断20是不是4和10的共同倍数。哪个正确？**
A. 是，20是4的5倍，但20不是10的倍数
B. 是，20是10的2倍，但20不是4的倍数
C. 不是，20太小了
D. 是，20是4的5倍，也是10的2倍
**答案：D**
| 选项 | 错因 |
|---|---|
| A | 矛盾 |
| B | 矛盾 |
| C | 太小错 |
| D | 正确 |

## Written

**7. 完整列倍数求5和12的最小公倍数。哪个步骤正确？**
A. 5的倍数：5、10、15、20、25、30、35、40、45、50、55、60……；12的倍数：12、24、36、48、60……；第一个共同倍数是30，所以最小公倍数是30
B. 5的倍数：5、10、15、20、25、30、35、40、45、50、55、60……；12的倍数：12、24、36、48、60……；第一个共同倍数是48，所以最小公倍数是48
C. 5的倍数：5、10、15、20、25、30、35、40、45、50、55、60……；12的倍数：12、24、36、48、60……；第一个共同倍数是60，所以最小公倍数是60
D. 5的倍数：5、10、15、20、25、30、35、40、45、50、55、60……；12的倍数：12、24、36、48、60……；第一个共同倍数是12，所以最小公倍数是12
**答案：C**
| 选项 | 错因 |
|---|---|
| A | 30不是12的倍数 |
| B | 48不是5的倍数 |
| C | 正确 |
| D | 12不是5的倍数 |

**8. 用除法检查结果是不是两个数的倍数。检查60是否是5和12的最小公倍数，哪个正确？**
A. 60÷5=12，60÷12=5，所以60是5和12的共同倍数
B. 60÷5=12，60÷12=5，所以60不是12的倍数
C. 60÷5=12，60÷12=6，所以60是最小公倍数
D. 60÷5=12，60÷12=5，但60不是最小公倍数
**答案：A**
| 选项 | 错因 |
|---|---|
| A | 正确 |
| B | 矛盾 |
| C | 商错 |
| D | 矛盾 |

## Application

**9. 两盏信号灯同时亮：一盏每6秒亮，一盏每10秒亮，几秒后第一次同时亮？**
A. 30秒后
B. 10秒后
C. 60秒后
D. 20秒后
**答案：A**
| 选项 | 错因 |
|---|---|
| A | 正确 |
| B | 不是6的倍数 |
| C | 是共同倍数但不是最小 |
| D | 不是6的倍数 |

**10. 说说最小公倍数怎样帮助通分。哪个说法正确？**
A. 最小公倍数帮助化简分数
B. 最小公倍数没有实际用途
C. 最小公倍数帮助找最大公因数
D. 通分时找两个分母的最小公倍数作为共同分母，这样分数大小不变
**答案：D**
| 选项 | 错因 |
|---|---|
| A | 混淆作用 |
| B | 无关 |
| C | 无关 |
| D | 正确 |
