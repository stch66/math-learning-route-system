# A10-3 乘法交换律

## Concept

**1. 乘法交换律说交换两个因数，积不变。7×8 和 8×7 在数组图里为什么一样？**
A. 因为先算了 7×8 才能算 8×7
B. 因为 7 和 8 都是一位数
C. 因为 7 比 8 小
D. 交换的是因数的顺序，数组的行和列互换但总数不变
**答案：D**
| 选项 | 错因 |
|---|---|
| A | 误以为有因果顺序关系 |
| B | 只看数字特征，没有理解交换律本质 |
| C | 只看大小，没有理解交换律本质 |
| D | 正确，理解乘法交换律的本质 |

**2. 变的是行和列的说法，不变的是（　）。**
A. 不变的是因数大小
B. 不变的是总数（积）
C. 变的是乘法
D. 变的是总数
**答案：B**
| 选项 | 错因 |
|---|---|
| A | 因数不变但非重点 |
| B | 正确 |
| C | 乘法本身变了 |
| D | 变化量错 |

## Model

**3. 画 6 行 9 列点阵，再把纸转一转看成 9 行 6 列。总数会变吗？**
A. 总数变了，要重新数
B. 总数变大，因为行数变了
C. 总数变小，因为列数变了
D. 总数不变，还是 54 个
**答案：D**
| 选项 | 错因 |
|---|---|
| A | 不懂交换 |
| B | 不需重数错 |
| C | 不懂交换 |
| D | 正确 |

**4. 用面积模型标出两个因数。哪个说法正确？**
A. 5×7 的面积比 7×5 大
B. 面积模型的方向决定了大小
C. 面积模型不能表示乘法
D. 5×7 和 7×5 的面积模型虽然摆放方向不同，但都是 35
**答案：D**
| 选项 | 错因 |
|---|---|
| A | 方向无关 |
| B | 方向决定大小错 |
| C | 误解模型 |
| D | 正确 |

## Mental

**5. 已知 6×8=48，不用重算说出 8×6=（　）。**
A. 28
B. 48
C. 14
D. 42
**答案：B**
| 选项 | 错因 |
|---|---|
| A | 算成加法 |
| B | 正确 |
| C | 进位错 |
| D | 算成减法 |

**6. 判断 5×9 和 9×5 是否相等。哪个说法正确？**
A. 不相等，因为 5×9 是 45，9×5 是 45
B. 不相等，因为因数位置变了
C. 不一定，要看哪个因数在前
D. 相等，因为乘法交换律
**答案：D**
| 选项 | 错因 |
|---|---|
| A | 自相矛盾，两个结果都是45却说不相等 |
| B | 不懂交换律 |
| C | 不懂交换律 |
| D | 正确，理解乘法交换律的本质 |

## Written

**7. 写一组乘法交换律等式，并标出两个因数。哪组最合适？**
A. 9×3=9×3
B. 9×3=3×9
C. 9×3=27=3×9
D. 9×3=27
**答案：B**
| 选项 | 错因 |
|---|---|
| A | 没有体现交换 |
| B | 正确 |
| C | 无意义重复 |
| D | 结构错误 |

**8. 把 9×3 改写成 3×9，说明什么时候这样更顺口。哪句最准确？**
A. 因为 3×9 比 9×3 更好看
B. 乘法交换律：两个因数交换位置，积不变；有时 3×9 更容易算
C. 因为老师要求这样写
D. 因为 9 比 3 大
**答案：B**
| 选项 | 错因 |
|---|---|
| A | 非数学理由 |
| B | 正确 |
| C | 非数学理由 |
| D | 非数学理由 |

## Application

**9. 6 行 9 列座位：按行数或列数说，座位总数变了吗？**
A. 总数不变，6×9=54 或 9×6=54
B. 按行数算更多
C. 按列数算更多
D. 要看先数哪边
**答案：A**
| 选项 | 错因 |
|---|---|
| A | 正确 |
| B | 误解模型 |
| C | 误解模型 |
| D | 不需重数 |

A. 7×5=35 和 7×5=35
B. 7×5=32 和 5×7=32
C. 7×5=35 和 5×7=35
D. 7×5=14 和 5×7=14
**答案：C**
| 选项 | 错因 |
|---|---|
| A | 没有体现交换 |
| B | 算错了 |
| C | 正确 |
| D | 算错了 |
