# A10-5 分配律直观理解

## Concept

**1. 分配律可以把 7×(10+5) 拆成 7×10+7×5。为什么拆开后面积不变？**
A. 因为分配的是同一个 7，面积只是分成了两部分，总面积不变
B. 因为 7×10 比 7×5 大
C. 因为拆开后 7 变大了
D. 因为 10+5=15，7×15 比较大
**答案：A**
| 选项 | 错因 |
|---|---|
| A | 正确 |
| B | 只看大小 |
| C | 方向错误 |
| D | 拆后变大错 |

**2. 说一说：大长方形被拆成了哪两块？**
A. 拆成宽 10 和宽 5 的两个小长方形，面积分别是 7×10 和 7×5
B. 拆成两个一样大的小长方形
C. 拆成高不一样的两个三角形
D. 拆成两个正方形
**答案：A**
| 选项 | 错因 |
|---|---|
| A | 正确 |
| B | 形状错误 |
| C | 形状错误 |
| D | 形状错误 |

## Model

**3. 画一个高 7、宽 10+5 的长方形，竖着分成 10 和 5 两段。哪个说法正确？**
A. 左边面积是 7×10=70，右边面积是 7×5=35，总面积 105
B. 左边面积是 7+10=17
C. 右边面积比左边大
D. 左边面积是 7×10=70，右边面积是 7×5=35，但总面积不确定
**答案：A**
| 选项 | 错因 |
|---|---|
| A | 正确 |
| B | 算成加法 |
| C | 总数不确定错 |
| D | 无关 |

**4. 在两块上分别写 7×10 和 7×5。哪个等式正确？**
A. 7×(10+5)=7×10+5
B. 7×(10+5)=70+35=105，和原来不一样
C. 7×(10+5)=7×10+7×5=105
D. 7×(10+5) 等于 7×10 减 7×5
**答案：C**
| 选项 | 错因 |
|---|---|
| A | 少乘了一个7 |
| B | 暗示不同 |
| C | 正确 |
| D | 把加法关系错看成减法 |

## Mental

**5. 用分配律算：9×20+9×4=（　）。**
A. 180
B. 216
C. 36
D. 24
**答案：B**
| 选项 | 错因 |
|---|---|
| A | 只算了9×20 |
| B | 正确 |
| C | 只算了9×4 |
| D | 只算了20+4 |

**6. 判断 7×15 可以拆成 7×10+7×5 吗？哪个正确？**
A. 不可以，因为 7×15 比 7×10 大
B. 不可以，因为 10+5 不等于 15
C. 可以，7×15=7×(10+5)=7×10+7×5=105
D. 可以，但结果会变小
**答案：C**
| 选项 | 错因 |
|---|---|
| A | 误解比较大 |
| B | 10+5=15是正确的 |
| C | 正确 |
| D | 拆后结果不变 |

## Written

**7. 完整写：8×(20+3)=8×20+8×3=（　）。**
A. 160
B. 184
C. 88
D. 24
**答案：B**
| 选项 | 错因 |
|---|---|
| A | 只算了8×20 |
| B | 正确 |
| C | 混淆运算 |
| D | 只算了8×3 |

**8. 把 6×12 拆成 6×10+6×2。哪个结果正确？**
A. 6×10+6×2=10+2=12
B. 6×10+6×2=60+2=62
C. 6×10+6×2=60+12=72
D. 6×10+6×2=6×12=72，但拆开不对
**答案：C**
| 选项 | 错因 |
|---|---|
| A | 方向错 |
| B | 矛盾 |
| C | 正确 |
| D | 加法错 |

## Application

**9. 长方形面积拆成两块：一块长方形地被分成两块，求总面积。哪步最简？**
A. 先算 8×11=88，再加 8=96
B. 直接算 8×19=152
C. 先算 8+11+3=22
D. 8×(11+3)=8×11+8×3=112 平方米
**答案：D**
| 选项 | 错因 |
|---|---|
| A | 加法错 |
| B | 虽然结果对但没有体现分配律 |
| C | 长宽误当加法 |
| D | 正确使用分配律 |

**10. 自己画一个能用分配律解释的面积图。哪道符合分配律？**
A. 宽 6，长 11+3，面积 (6×11)×(6×3)
B. 宽 6，长 11+3，面积 6×(11+3)=6×11+6×3=84
C. 宽 6，长 11+3，面积 6×14=84，但没有写出分配律
D. 宽 6，长 11+3，面积 6+11+3=20
**答案：B**
| 选项 | 错因 |
|---|---|
| A | 乘法错 |
| B | 正确 |
| C | 加法错 |
| D | 结果对但无关 |
