# A10-20 3的整除规则

## Concept

**1. 3的整除规则：各位数字和能被3整除。为什么不用真的除也能先判断？**
A. 因为3的倍数都是3的倍数
B. 因为3是一位数
C. 因为只把各位数字加起来看和是否被3整除，不用算除法
D. 因为3比较小
**答案：C**
| 选项 | 错因 |
|---|---|
| A | 循环论证，没有解释规则 |
| B | 没有解释规则本质 |
| C | 正确 |
| D | 没有解释规则本质 |

**2. 判断一个数能否被3整除，要看哪里？比如123这一类数。**
A. 看是不是3的倍数
B. 看十位
C. 看各位数字之和
D. 看个位
**答案：C**
| 选项 | 错因 |
|---|---|
| A | 没有说明具体判断方法 |
| B | 混淆了2/5/10的规则 |
| C | 正确 |
| D | 混淆了2/5/10的规则 |

## Model

**3. 把423写在数位表里，圈出需要看的数字。423能否被3整除？**
A. 不能确定
B. 能，个位是3
C. 能，4+2+3=9，9能被3整除
D. 能，423÷3=141
**答案：C**
| 选项 | 错因 |
|---|---|
| A | 规则可以直接判断 |
| B | 混淆个位规则 |
| C | 正确，用数字和判断 |
| D | 结果对，但不是按规则判断 |

**4. 用分组图表示能整除时没有余数。423个物品每3个一组，分了几组？**
A. 分了141组
B. 分了420组
C. 分了139组
D. 分了142组
**答案：A**
| 选项 | 错因 |
|---|---|
| A | 正确，423÷3=141 |
| B | 423-3=420，没有理解除法 |
| C | 商偏小 |
| D | 商偏大 |

## Mental

**5. 快速判断：423能否被3整除；173能否被3整除；369能否被3整除。哪个正确？**
A. 423能，173能，369能
B. 423能，173不能，369能
C. 423能，173能，369不能
D. 423不能，173能，369能
**答案：B**
| 选项 | 错因 |
|---|---|
| A | 173判断错 |
| B | 正确 |
| C | 173和369判断错 |
| D | 173漏判 |

**6. 说出你的判断依据，不列竖式。判断173能否被3整除，依据是？**
A. 173是奇数，所以不能
B. 1+7+3=11，11是3的倍数，所以173能
C. 个位是3，所以不能
D. 1+7+3=11，11不能被3整除，所以173不能
**答案：D**
| 选项 | 错因 |
|---|---|
| A | 奇偶与3整除无关 |
| B | 11÷3=3余2，不是3的倍数 |
| C | 3的规则不看个位 |
| D | 正确 |

## Written

**7. 选择423和173，写出判断过程。哪个过程正确？**
A. 423：4+2+3=9，能被3整除；173：个位是3，不能被3整除
B. 423：4+2+3=9，能被3整除；173：1+7+3=11，不能被3整除
C. 423：4+2+3=9，不能被3整除；173：1+7+3=11，能被3整除
D. 423：4+2+3=9，能被3整除；173：1+7+3=11，能被3整除
**答案：B**
| 选项 | 错因 |
|---|---|
| A | 173用了个位规则 |
| B | 正确 |
| C | 173判断反了 |
| D | 173判断反了 |

**8. 改错题：同学说“714 的个位是4，所以不能被3整除”。怎样改最准确？**
A. 应该把各位数字相加，7+1+4=12，12能被3整除，所以714能被3整除
B. 应该看十位，十位是1，所以714不能被3整除
C. 个位是4确实能决定能否被3整除
D. 714是偶数，所以一定不能被3整除
**答案：A**
| 选项 | 错因 |
|---|---|
| A | 正确，用各位数字和判断 |
| B | 看错数位 |
| C | 3的整除规则不看个位 |
| D | 奇偶与3的整除规则无关 |

## Application

**9. 三人一组分队：编号546能不能按3个一组正好分完？哪个最符合3的整除规则？**
A. 不能，因为546是偶数
B. 能，因为546÷3=182
C. 能，因为5+4+6=15能被3整除
D. 能，因为546是3的倍数
**答案：C**
| 选项 | 错因 |
|---|---|
| A | 奇偶与3整除无关 |
| B | 结果对，但不是按规则判断 |
| C | 正确，用各位数字和判断 |
| D | 判断对但没有说明规则 |

**10. 自己写一个能被3整除和一个不能整除的数。参考369和173，哪个正确？**
A. 能被3整除：369（3+6+9=18）；不能被3整除：173（1+7+3=11）
B. 能被3整除：258（2+5+8=15）；不能被3整除：372（3+7+2=12）
C. 能被3整除：357（3+5+7=15）；不能被3整除：168（1+6+8=15）
D. 能被3整除：444（4+4+4=12）；不能被3整除：255（2+5+5=12）
**答案：A**
| 选项 | 错因 |
|---|---|
| A | 正确，369和是18能整除，173和是11不能整除 |
| B | 258和是15能，372和是12也能，互换了 |
| C | 357和是15能，168和是15也能，两个都能 |
| D | 444和是12能，255和是12也能，两个都能 |
