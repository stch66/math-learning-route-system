# A19-4.16 小数补0比较 选择题（按五层设计）

## 概念层（2题）

**第1题**：小数末尾添 0 后，大小会怎样？
- A. 不变（对：小数末尾 0 不改变位值）
- B. 一定变大（错：3.4=3.40）
- C. 一定变小（错：大小不变）
- D. 变成负数（错：添 0 不改变符号）

答案：A

---

**第2题**：3.7 可以写成哪一个等值小数？
- A. 0.37（错：缩小 10 倍）
- B. 3.70（对：末尾添 0 大小不变）
- C. 3.07（错：位值改变）
- D. 37.0（错：扩大 10 倍）

答案：B

---

## 模型层（2题）

**第3题**：比较 5.4 和 5.38，补 0 后应看成什么？
- A. 5.04 和 5.38（错：补 0 位置错）
- B. 0.54 和 5.38（错：移动小数点了）
- C. 5.40 和 5.38（对：补到相同位数再比）
- D. 54 和 538（错：删小数点改变位值）

答案：C

---

**第4题**：在位值表里，6.2 和 6.20 的关系是什么？
- A. 6.20 更大（错：末尾 0 不增大）
- B. 6.2 更大（错：二者相等）
- C. 不能比较（错：可以比较且相等）
- D. 同一个数（对：百分位补 0 不改变大小）

答案：D

---

## 快速反应层（2题）

**第5题**：快速判断：哪个和 0.5 相等？
- A. 0.50（对：末尾添 0 不变）
- B. 0.05（错：位值变小）
- C. 5.0（错：等于 5）
- D. 0.55（错：大于 0.5）

答案：A

---

**第6题**：哪个比较式正确？
- A. 2.30=23（错：小数点不能删）
- B. 2.30=2.3（对：末尾 0 可省略）
- C. 2.30<2.3（错：二者相等）
- D. 2.30>2.3（错：二者相等）

答案：B

---

## 规范层（2题）

**第7题**：小数补 0 后，哪一行比较过程最规范？
- A. 4.8=4.75，因为整数部分相同（错：小数部分不同）
- B. 把 4.75 改成 4.7500 才能变大（错：补 0 不改变大小）
- C. 4.8=4.80，4.75<4.80，所以 4.75<4.8（对：补 0 后按位比较）
- D. 4.8<4.75，因为 75>8（错：没有按位值比较）

答案：C

---

**第8题**：学生把 1.20 判成大于 1.2，错因是什么？
- A. 把 1 看成整数（错：整数部分不是错因）
- B. 不会写百分位（错：主要是等值理解错）
- C. 把小数变成分数（错：不是分数问题）
- D. 把末尾 0 当成增加了数值（对：末尾 0 不改变大小）

答案：D

---

## 应用层（2题）

**第9题**：价格牌 2.50 元和 2.5 元表示什么关系？
- A. 金额相同（对：2.50=2.5）
- B. 2.50 元更贵（错：末尾 0 不改变金额）
- C. 2.5 元更贵（错：二者相等）
- D. 不能作为价格（错：都可表示价格）

答案：A

---

**第10题**：测量 0.300 米和 0.3 米，哪句话正确？
- A. 0.300 是负数（错：没有负号）
- B. 长度相同，但 0.300 显示到千分位（对：数值相等，精确记录位数不同）
- C. 0.300 更长（错：数值相等）
- D. 0.3 更长（错：数值相等）

答案：B
