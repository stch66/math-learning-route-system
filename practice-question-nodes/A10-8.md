# A10-8 估算和

## Concept

**1. 估算和要先把加数变成好算的近似数。367+204 可以约成多少加多少？**
A. 约成 367+204
B. 约成 360+210
C. 约成 370+204
D. 约成 400+200
**答案：D**
| 选项 | 错因 |
|---|---|
| A | 没有进行估算 |
| B | 360+210没有遵循四舍五入原则 |
| C | 204约成204没有体现估算简化原则 |
| D | 正确，367≈400，204≈200，便于计算 |

**2. 估算不是精算，答案应该说明"大约"。哪个估算结果最合理？**
A. 367+204≈400+200=600，所以和大约是 600
B. 367+204≈571
C. 367+204≈400+204=604
D. 367+204≈367+204=571
**答案：A**

## Model

**3. 在数轴上分别标出 482 和 217 的近似数。哪个正确？**
A. 482 约到 500，217 约到 220
B. 482 约到 500，217 约到 200
C. 482 约到 480，217 约到 217
D. 482 约到 482，217 约到 200
**答案：B**

**4. 画两个条形，表示两次人数合起来的大概总量。估算和大约是多少？**
A. 360+210=570 人
B. 400+200=600 人
C. 400+204=604 人
D. 367+204=571 人
**答案：B**

## Mental

**5. 快速估算：638+192≈（　）。**
A. 600
B. 830
C. 800
D. 200
**答案：C**
| 选项 | 错因 |
|---|---|
| A | 600只是638的近似值，不是估算和 |
| B | 830是精确值，不是估算 |
| C | 正确，638+192≈600+200=800 |
| D | 200只是192的近似值，不是估算和 |

**6. 判断精确和会接近 600 还是 1000？哪个正确？**
A. 接近 1000，因为 367+204=571，600 是 1000 的一半
B. 接近 600，因为 367+204=571，571 最接近 600
C. 接近 1000，因为 400+200=600
D. 接近 600，因为 367+204=571，571 小于 600
**答案：B**
| 选项 | 错因 |
|---|---|
| A | 未理解估算 |
| B | 正确 |
| C | 误解接近 |
| D | 判断对但理由错 |

## Written

**7. 写出估算过程：529≈（　），184≈（　），所以和约（　）。**
A. 529≈500，184≈200，和约 500
B. 529≈500，184≈200，和约 713
C. 529≈500，184≈200，和约 700
D. 529≈530，184≈180，和约 710
**答案：C**
| 选项 | 错因 |
|---|---|
| A | 只写了第一个近似数 |
| B | 写成了精确和 |
| C | 正确 |
| D | 不是约到整百的估算 |

**8. 再精算一次，看看是否落在合理范围内。哪个结果正确？**
A. 367+204=571，离 600 很远，不合理
B. 367+204=571，离 600 很近，合理
C. 367+204=571，和 600 差 29，很合理
D. 367+204=571，比 600 小很多，不合理
**答案：B**
| 选项 | 错因 |
|---|---|
| A | 判断反 |
| B | 正确 |
| C | 判断错 |
| D | 精确值错 |

## Application

**9. 两次活动人数：第一次 458 人，第二次 193 人。大约一共多少人？**
A. 约 651 人，直接算
B. 约 500 人，只看第一个数
C. 约 700 人，458+193≈500+200=700
D. 约 200 人，只看第二个数
**答案：C**
| 选项 | 错因 |
|---|---|
| A | 直接算是精算不是估算 |
| B | 只看一个 |
| C | 正确，458≈500，193≈200，和约700 |
| D | 只看一个 |

**10. 自己编一道先估算和再检查的题。哪个估算最合理？**
A. 289+405≈289+405=694
B. 289+405≈280+410=690
C. 289+405≈290+405=695
D. 289+405≈300+400=700
**答案：D**
| 选项 | 错因 |
|---|---|
| A | 精确值 |
| B | 部分约 |
| C | 精确值 |
| D | 正确 |
