# A10-21 9的整除规则

## Concept

**1. 9的整除规则：各位数字和能被9整除。为什么不用真的除也能先判断？**
A. 因为9是一位数
B. 因为只把各位数字加起来看和是否被9整除，不用算除法
C. 因为9的倍数都是9的倍数
D. 因为9比较小
**答案：B**
| 选项 | 错因 |
|---|---|
| A | 没有解释规则本质 |
| B | 正确 |
| C | 循环论证 |
| D | 没有解释规则本质 |

---

**2. 判断一个数能否被9整除，要看哪里？比如234这一类数。**
A. 看十位
B. 看各位数字之和
C. 看个位
D. 看是不是9的倍数
**答案：B**
| 选项 | 错因 |
|---|---|
| A | 混淆了2/5/10的规则 |
| B | 正确 |
| C | 混淆了2/5/10的规则 |
| D | 没有说明具体判断方法 |

---

## Model

**3. 把342写在数位表里，圈出需要看的数字。342能否被9整除？**
A. 不能确定
B. 能，3+4+2=9，9能被9整除
C. 能，个位是2
D. 能，342÷9=38
**答案：B**
| 选项 | 错因 |
|---|---|
| A | 规则可以直接判断 |
| B | 正确，用各位数字和判断 |
| C | 9的规则不看个位 |
| D | 结果对，但不是按规则判断 |

---

**4. 用分组图表示能整除时没有余数。342个物品每9个一组，分了几组？**
A. 分了38组
B. 分了9组
C. 分了333组
D. 分了37组
**答案：A**
| 选项 | 错因 |
|---|---|
| A | 正确，342÷9=38 |
| B | 误把除数当商 |
| C | 342-9=333，没有理解除法 |
| D | 商偏小 |

---

## Mental

**5. 快速判断：459能否被9整除；527能否被9整除；702能否被9整除。哪个正确？**
A. 459不能，527能，702能
B. 459能，527能，702不能
C. 459能，527能，702能
D. 459能，527不能，702能
**答案：D**
| 选项 | 错因 |
|---|---|
| A | 459和527判断反 |
| B | 527和702判断反 |
| C | 527判断错 |
| D | 正确 |

---

**6. 说出你的判断依据，不列竖式。判断527能否被9整除，依据是？**
A. 5+2+7=14，14能被9整除，所以527能
B. 527是奇数，所以能
C. 个位是7，所以能
D. 5+2+7=14，14不能被9整除，所以527不能
**答案：D**
| 选项 | 错因 |
|---|---|
| A | 14不能被9整除 |
| B | 奇偶与9整除无关 |
| C | 9的规则不看个位 |
| D | 正确 |

## Written

**7. 选择459和527，写出判断过程。哪个过程正确？**
A. 459：个位是9，能被9整除；527：个位是7，能被9整除
B. 459：4+5+9=18，能被9整除；527：5+2+7=14，不能被9整除
C. 459：4+5+9=18，能被9整除；527：5+2+7=14，能被9整除
D. 459：4+5+9=18，不能被9整除；527：5+2+7=14，能被9整除
**答案：B**
| 选项 | 错因 |
|---|---|
| A | 用个位判断是错误规则 |
| B | 正确 |
| C | 527判断反了 |
| D | 两个都判断反了 |

---

**8. 改错题：同学说“936 的个位是6，所以不能被9整除”。怎样改最准确？**
A. 应该把各位数字相加，9+3+6=18，18能被9整除，所以936能被9整除
B. 应该看十位，十位是3，所以936不能被9整除
C. 个位是6确实能决定能否被9整除
D. 936是偶数，所以一定不能被9整除
**答案：A**
| 选项 | 错因 |
|---|---|
| A | 正确，用各位数字和判断 |
| B | 看错数位 |
| C | 9的整除规则不看个位 |
| D | 奇偶与9的整除规则无关 |

## Application

**9. 九宫格分组：编号684能不能按9个一组正好分完？哪个最符合9的整除规则？**
A. 能，因为684÷9=76
B. 不能，因为684是偶数
C. 能，因为6+8+4=18能被9整除
D. 能，因为684是9的倍数
**答案：C**
| 选项 | 错因 |
|---|---|
| A | 结果对，但不是按规则判断 |
| B | 奇偶与9整除无关 |
| C | 正确，用各位数字和判断 |
| D | 判断对但没有说明规则 |

---

**10. 自己写一个能被9整除和一个不能整除的数。参考378和472，哪个正确？**
A. 能被9整除：378（3+7+8=18）；不能被9整除：472（4+7+2=13）
B. 能被9整除：523（5+2+3=10）；不能被9整除：438（4+3+8=15）
C. 能被9整除：297（2+9+7=18）；不能被9整除：351（3+5+1=9）
D. 能被9整除：459（4+5+9=18）；不能被9整除：450（4+5+0=9）
**答案：A**
| 选项 | 错因 |
|---|---|
| A | 正确，378和是18能整除，472和是13不能整除 |
| B | 438和是15也能整除，互换了 |
| C | 351也能被9整除 |
| D | 450也能被9整除 |
