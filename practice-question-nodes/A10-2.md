# A10-2 加法结合律

## Concept

**1. 加法结合律改变的是先把哪两个数合起来。48+67+23 中先算哪两个更方便？**
A. 先算 67+23，因为 67+23=90
B. 随便先算哪两个，结果一样
C. 先算 48+23，因为数字小
D. 先算 48+67，因为能凑整
**答案：A**
| 选项 | 错因 |
|---|---|
| A | 正确，67+23=90，90+48=138 |
| B | 虽然结果一样，但未体现"更方便" |
| C | 虽然结果一样，但未体现"更方便" |
| D | 48+67=115，没有凑整 |

**2. 为什么加括号会改变顺序，但不改变总和？**
A. 因为把两个数先加起来，这两个数就变小了
B. 因为减法有结合律，加法没有
C. 因为把两个数先加起来，这两个数就变大了
D. 加括号只是改变运算顺序，和取决于所有数相加的总和
**答案：D**
| 选项 | 错因 |
|---|---|
| A | 误解结合律原理，加数不会因为先加而变小 |
| B | 与结合律无关 |
| C | 与结合律无关 |
| D | 正确，加括号只改变运算顺序，总和不变 |

## Model

**3. 画三盒彩笔 35、29、41 支，圈出先合并的两盒。哪两盒先合并更方便？**
A. 先合并 35 和 29，凑成 64，再加上 41
B. 三盒必须一起加，不能分开
C. 先合并 35 和 41，凑成 76，再加上 29
D. 先合并 29 和 41，凑成 70，再加上 35
**答案：D**
| 选项 | 错因 |
|---|---|
| A | 可以先合并，但64不如70好算 |
| B | 必须一起加错，结合律允许先合并不影响结果 |
| C | 可以先合并，但76不如70好算 |
| D | 正确，29+41=70，再加35更方便 |

**4. 用括号表示两种分组：(35+29)+41 和 35+(29+41)。哪个说法正确？**
A. 两种分组的结果都是 105，总和不变
B. (35+29)+41 比 35+(29+41) 大
C. 35+(29+41) 比 (35+29)+41 大
D. 两种分组结果不同
**答案：A**
| 选项 | 错因 |
|---|---|
| A | 正确，结合律的本质 |
| B | 误以为不同分组结果不同 |
| C | 误以为不同分组结果不同 |
| D | 误以为不同分组结果不同 |

## Mental

**5. 快速算：57+43+18=（　），先找能凑整的一组。**
A. 100
B. 118
C. 108
D. 115
**答案：B**
| 选项 | 错因 |
|---|---|
| A | 100只是57+43，部分和不是最终答案 |
| B | 正确，57+43=100，100+18=118 |
| C | 算术错 |
| D | 算术错 |

**6. 判断 57+43+18 先算 57+43 为什么更快。哪个说法最准确？**
A. 因为老师要求先算前两个
B. 因为 57+43=100，100+18 更容易算
C. 因为 43+18 比 57+18 大，大的先加比较快
D. 因为 57+43+18 的顺序可以随便改
**答案：B**
| 选项 | 错因 |
|---|---|
| A | 非数学理由，服从权威 |
| B | 正确，凑整后计算最简便 |
| C | 只看大小，没有找凑整 |
| D | 虽然正确但未体现"更方便" |

## Written

**7. 用结合律写出简便计算过程。哪步最准确？**
A. 64+36+29=(64+36)+29=100+29=129
B. 64+36+29=64+36+29=129
C. 64+36+29=64+(36+29)=64+65=129
D. 64+36+29=(64+29)+36=93+36=129
**答案：A**
| 选项 | 错因 |
|---|---|
| A | 正确，凑整后计算最简便 |
| B | 没有体现结合律 |
| C | 虽然正确但不是最简便 |
| D | 虽然正确但不是最简便 |

**8. 圈出你改变的括号，并写"总和不变"。改变后哪个括号位置是正确的？**
A. (64+36)+29 和 64+36+29 意思不同
B. (64+36)+29 或 64+(36+29)
C. (64+36)+29 和 (64+36+29) 一样
D. 括号必须加在 64+29 周围
**答案：B**
| 选项 | 错因 |
|---|---|
| A | 意思不同错，两种括号位置都符合结合律 |
| B | 正确，结合律允许两种不同的括号位置 |
| C | 意思相同对但结构不精确 |
| D | 没有数学依据 |

## Application

**9. 三盒彩笔先凑整：三盒共多少支？请用结合律算。哪步最简？**
A. 58+27=85，85+33=118
B. (58+33)+27=91+27=118，共 118 支
C. 58+33+27=118，直接算
D. 33+27=60，60+58=118
**答案：D**
| 选项 | 错因 |
|---|---|
| A | 未凑整，不是最简便 |
| B | 58+33=91，没有凑整 |
| C | 直接算没有体现结合律 |
| D | 正确，33+27=60，60+58=118 |

**10. 自己找三个数，让其中两个能凑整。哪个组合最好？**
A. 44、56、31（44+56=100）
B. 7、18、25（7+18=25）
C. 22、33、45（22+33=55，不是整十）
D. 15、25、35（15+25=40）
**答案：A**
| 选项 | 错因 |
|---|---|
| A | 正确，44+56=100能凑整 |
| B | 7+18=25，不是整十整百 |
| C | 22+33=55，不是整十整百 |
| D | 15+25=40能凑整，但A选项更典型（凑成100）|
