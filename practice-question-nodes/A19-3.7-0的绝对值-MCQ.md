# A19-3.7 0的绝对值 选择题（按五层设计）

## 概念层（2题）

**第1题**：0 的绝对值是多少？
- A. 不存在（错：0 有绝对值）
- B. 0（对：0 到 0 的距离是 0）
- C. 1（错：没有 1 个单位距离）
- D. -1（错：绝对值不为负）

答案：B

---

**第2题**：为什么 |0| 不等于 1？
- A. 因为 0 没有数轴位置（错：0 是原点）
- B. 因为绝对值都等于 1（错：说法错误）
- C. 因为 0 到 0 没有距离（对：|0| 表示距离 0）
- D. 因为 0 是负数（错：0 不是负数）

答案：C

---

## 模型层（2题）

**第3题**：数轴上原点对应的绝对值模型是什么？
- A. 从 0 到 1 的长度为 1（错：那是 |1|）
- B. 从 0 到 -1 的长度为 -1（错：距离不能为负）
- C. 原点不能参与模型（错：原点正是基准）
- D. 从 0 到 0 的长度为 0（对：原点到自身距离为 0）

答案：D

---

**第4题**：若点 A 与原点重合，|A| 应读作多少？
- A. 0（对：点在原点，距离为 0）
- B. A（错：A 是点名，不是读数）
- C. 1（错：没有离开原点）
- D. -0（错：-0 数值仍是 0，但写法不规范）

答案：A

---

## 快速反应层（2题）

**第5题**：快速判断：哪个数的绝对值等于 0？
- A. 所有数（错：大多数数到 0 有距离）
- B. 只有 0（对：只有 0 到 0 的距离为 0）
- C. 0 和 1（错：|1|=1）
- D. 0 和 -1（错：|-1|=1）

答案：B

---

**第6题**：下面哪条等式正确？
- A. |0|=-1（错：绝对值不为负）
- B. |0| 不存在（错：0 有绝对值）
- C. |0|=0（对：0 的绝对值为 0）
- D. |0|=1（错：0 到 0 距离不是 1）

答案：C

---

## 规范层（2题）

**第7题**：把“绝对值不为负”说完整，哪项最准确？
- A. 绝对值一定是正数（错：|0| 不是正数）
- B. 绝对值可以是负数（错：距离不能为负）
- C. 绝对值只适用于负数（错：正数和 0 也有绝对值）
- D. 绝对值可以是 0，也可以是正数（对：|0|=0，非零数绝对值为正）

答案：D

---

**第8题**：学生说“0 没有绝对值，因为没有方向”。怎样纠正？
- A. 绝对值看距离，0 到 0 的距离是 0（对：方向不是绝对值的必要条件）
- B. 他说得对，0 不能放在数轴上（错：0 是原点）
- C. 0 的绝对值是 -0.5（错：无依据）
- D. 只有正数有绝对值（错：负数和 0 也有绝对值）

答案：A

---

## 应用层（2题）

**第9题**：账户本月变化 0 元，若只问变化幅度，应是多少？
- A. 无法表示（错：0 可以表示无变化）
- B. 0 元（对：没有变化，绝对值为 0）
- C. +1 元（错：题目没有增加 1 元）
- D. -1 元（错：题目没有减少 1 元）

答案：B

---

**第10题**：海拔 0 米处到海平面的距离是多少？
- A. -1 米（错：距离不能为负）
- B. 无法测量（错：可以测量为 0）
- C. 0 米（对：海拔 0 米就在海平面）
- D. 1 米（错：没有离开海平面）

答案：C
