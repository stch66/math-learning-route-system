# A10-18 5的整除规则

## Concept

**1. 5的整除规则：个位是0或5。为什么不用真的除也能先判断？**
A. 因为5的倍数比较好算
B. 因为5的倍数个位都是0或5
C. 因为5是一位数
D. 因为只由个位决定，不用看其他数位
**答案：B**
| 选项 | 错因 |
|---|---|
| A | 只说好算，没有解释规则本质 |
| B | 正确 |
| C | 循环论证 |
| D | 说明了看个位，但没有解释为什么是0或5 |

**2. 判断一个数能否被5整除，要看哪里？比如70这一类数。**
A. 看所有数位之和
B. 看是不是5的倍数
C. 看个位
D. 看十位
**答案：C**
| 选项 | 错因 |
|---|---|
| A | 5的整除规则只与个位相关 |
| B | 没有说明具体判断方法 |
| C | 正确 |
| D | 十位与5的整除规则无关 |

## Model

**3. 把95写在数位表里，圈出需要看的数字。95能否被5整除？**
A. 能，个位是5
B. 不能确定
C. 能，十位是9
D. 能，9+5=14
**答案：A**
| 选项 | 错因 |
|---|---|
| A | 正确 |
| B | 看错数位 |
| C | 混淆了其他规则 |
| D | 不理解规则 |

**4. 用分组图表示能整除时没有余数。95个物品每5个一组，分了几组？**
A. 分了90组
B. 分了100组
C. 分了19组
D. 分了95组
**答案：C**
| 选项 | 错因 |
|---|---|
| A | 误以为95-5就是组数 |
| B | 误以为加5就是组数 |
| C | 正确，95÷5=19 |
| D | 误以为每组1个 |

## Mental

**5. 快速判断：95能否被5整除；72能否被5整除；230能否被5整除。哪个正确？**
A. 95能，72能，230不能
B. 95能，72能，230能
C. 95能，72不能，230能
D. 95不能，72能，230能
**答案：C**
| 选项 | 错因 |
|---|---|
| A | 72和230判断反 |
| B | 72个位是2漏判 |
| C | 正确 |
| D | 72漏判 |

**6. 说出你的判断依据，不列竖式。判断72能否被5整除，依据是？**
A. 个位是2，大于0，所以能
B. 十位是7，所以能
C. 72是偶数，所以不能
D. 个位是2，不是0或5，所以不能
**答案：D**
| 选项 | 错因 |
|---|---|
| A | 个位是2不能被5整除 |
| B | 误解规则 |
| C | 偶数与5的整除规则无关 |
| D | 正确 |

## Written

**7. 选择95和72，写出判断过程。哪个过程正确？**
A. 95个位是5，能被5整除；72个位是2，不能被5整除
B. 95个位是4，不能被5整除；72个位是8，能被5整除
C. 95个位是4，能被5整除；72个位是8，能被5整除
D. 95个位是5，能被5整除；72个位是2，也能被5整除
**答案：A**
| 选项 | 错因 |
|---|---|
| A | 正确 |
| B | 95个位不是4，72也不能被5整除 |
| C | 95个位是5不是4，72也不能被5整除 |
| D | 72个位是2，不能被5整除 |

**8. 改错题：同学说“415 的十位是1，所以不能被5整除”。怎样改最准确？**
A. 应该看个位，个位是5，所以415能被5整除
B. 应该看百位，百位是4，所以415不能被5整除
C. 十位是1确实能决定能否被5整除
D. 415是奇数，所以一定不能被5整除
**答案：A**
| 选项 | 错因 |
|---|---|
| A | 正确，用个位0或5判断 |
| B | 看错数位 |
| C | 十位不能决定5的整除 |
| D | 奇偶与5的整除规则无关 |

## Application

**9. 五人一组排队：编号140能不能按5个一组正好分完？哪个最符合5的整除规则？**
A. 不能，因为140是偶数
B. 能，因为140是5的28倍
C. 能，因为140÷5=28
D. 能，因为140个位是0
**答案：D**
| 选项 | 错因 |
|---|---|
| A | 偶数不影响能否被5整除 |
| B | 结果对，但没有体现个位规则 |
| C | 结果对，但不是按规则判断 |
| D | 正确，用个位是0判断 |

**10. 自己写一个能被5整除和一个不能整除的数。参考70和83，哪个正确？**
A. 能被5整除：70；不能被5整除：83
B. 能被5整除：67；不能被5整除：75
C. 能被5整除：63；不能被5整除：72
D. 能被5整除：75；不能被5整除：70
**答案：A**
| 选项 | 错因 |
|---|---|
| A | 正确，70个位0，83个位3 |
| B | 67个位7不能被5整除 |
| C | 63个位3不能被5整除 |
| D | 70个位0，也能被5整除 |
